Bridges

Bridges is a web and mobile application made to help hopeful immigrants coming to the US pass the US Immigration test. The team used a trivia type game to display questions incorporated a translation API to translate each question into the users' native language. The team made this project after realizing that there were insufficient resources to help hopefuls pass the immigration test. Technologies used: Materialize, Firebase, Yanslate API, RSS Feed and more!
